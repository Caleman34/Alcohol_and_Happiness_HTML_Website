Do happier countries drink more or less than sad countries?

A University of Texas at San Antonio Data Analytics Bootcamp Project

The purpose of this project was to analyze if there is a relationship between happiness and alcohol consumption. To accomplish this analysis, we used multiple data sets on 111 countries in 9 different regions across the world in 2016 from kaggle.com.

After assembling the data using pandas, we used Matplotlib and Seaborn to graph and chart various variables such as GDP, Happiness Score, Alcohol Consumption, Perception of Corruption, Social Support. This site provides the souce data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations observed.
